DESCRIPCION DEL PROYECTO


El sistema permitira a los usuarios gestionar sus tareas o objetivos que tienen
que realizar o tambien es se puede utilizar como un recordatorio de las cosas
que tienen pendiente.

objetivos principales:

1. crear tareas con un titulo y descripcion.
2. Marcar tareas como completadas.
3. Editar el contenido de las tareas.
4. Eliminar tareas.
5. Visualizar todas las tareas en una lista.

Estructura del proyecto:
FRONTEND:

HTML: Estructura de la pagina.
CSS : Estilos y el diseño.
JavaScript : Validacion de formulario.

BACKEND:

PHP : Manejo de las solicitudes para agregar,leer,actualizar y eliminar.

BASE DE DATOS : Usare phpAdmin para crear la base de datos llamada "GestionTareas"
la tablas que estare utilizando son tres.

1.Usuarios.
2.Tareas.
3.Cat
egorias.
